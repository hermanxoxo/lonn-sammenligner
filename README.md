# Lønnsammenligner

En enkel webapp for å sammenligne lønn mellom Herman og Ditlev.

## Funksjoner

- Klikk på et bilde for å registrere brutto lønn (time-, måneds- eller årslønn)
- All lønn konverteres og vises som årslønn og månedslønn
- Lønnshistorikk lagres i nettleserens `localStorage` — data forsvinner ikke ved refresh
- Sammenligning nederst viser hvem som leder, differanse i kroner og prosent

## Hvordan kjøre

Åpne `index.html` direkte i nettleseren. Ingen installasjon eller server nødvendig.
